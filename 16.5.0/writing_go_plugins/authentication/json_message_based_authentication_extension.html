Authentication plugin - JSON API - Message based

The objective of this guide is to explain how to write a authentication plugin, for Go.

Useful references:

  • Overview of authentication plugins - External link to Go's user documentation
  • Overview of message-based APIs
  • Structure of a plugin and writing one
  • A sample authentication plugin - sample authenticator

An authentication plugin is a Go plugin, which claims to support to extension name authentication in its identifier, and responds appropriately to the messages mentioned below. It's probably easiest to learn from the sample plugin mentioned above.
